_Classification must follow formal markings and dissemination controls (e.g. For LE Use Only)._

## Criminal offence data governance (GDPR Art. 10)

* Processing strictly limited to legitimate law enforcement purposes.
* Access limited to specially authorised personnel.
* Dissemination requires case-level justification and logging.
* Metadata must reflect source system and legal authority.

## Chain of custody expectations

Where Flowintel is used for storing investigative material:
* Every evidence upload must be logged with:
  * User ID
  * Timestamp
  * Source description
  * Hash value (if applicable)
* Modifications prohibited; only new versions appended.
* Export actions must produce:
  * Integrity hash
  * Responsible officer ID
  * Date/time stamp
  * Original object reference

## Retention & deletion exceptions

Retention may exceed standard terms where:
* A court order requires preservation
* Data are relevant to ongoing legal action
* Statutory minimum retention periods apply for criminal records

Exceptions must be recorded in:
* Case file
* Retention ledger
* DPIA addendum

## Operational access controls

* Access requires explicit case assignment.
* Periodic access review at defined intervals.
* Revocation at end of mission, investigation, or reassignment.
