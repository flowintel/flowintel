Repository actions
Check — verify the local path and validate manifest.json

Edit — update the repository name, URL or local path
Delete — remove the repository from Flowintel

Diff — compare the repository version with your local copy
Refresh — re-import to update an already-imported template
Import — add this template to Flowintel for the first time
